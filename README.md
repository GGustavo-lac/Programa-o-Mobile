# Mobile Programming Project Template

## Description
This project is a mobile application that aims to provide an exceptional user experience through modern design and robust functionality. The application resolves real-world problems by leveraging mobile technology.

## Technologies
- React Native
- TypeScript
- Redux
- Axios
- Firebase for backend

## Features
- User authentication with Firebase
- Data fetching using Axios
- State management with Redux
- Responsive design for various screen sizes
- Push notifications

## Installation
To get started with the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/GGustavo-lac/Programa-o-Mobile.git
   cd Programa-o-Mobile
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To run the application on your device or simulator:
1. Make sure you have all necessary dependencies installed (Node.js, npm, Android Studio or Xcode).
2. Run the following command:
   ```bash
   npm start
   ```
3. Follow the instructions to open the app in your simulator or connected device.
